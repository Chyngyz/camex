# Инструкция по настройке проекта
Установить **npm пакеты**
```sh
$ npm install
```
Установить **bower dependencies**
```sh
$ bower install
```
Поднять **локальный сервер**
```sh
$ gulp serve
```
Собрать **build (соберется в папке dist)**
```sh
$ gulp
```
Поднять **локальный сервер на dist**
```sh
$ gulp serve:dist
```
